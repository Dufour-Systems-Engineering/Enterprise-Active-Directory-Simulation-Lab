# Future Delegation Expansion
**Last Updated:** May 24, 2026
**Domain:** lab.local

## Current Foundation
The successfully implemented two-tier delegation model (Help Desk Tier 1 vs. IT Admins Tier 2), combined with strong AGDLP group structures and a thoroughly tested isolated Test OU, provides a solid platform for continued development.

## Planned Next Phase
I plan to extend the current model with the following enhancements:

### Manager-Level Self-Service Delegation
Grant department managers (`GG_Managers`) limited rights to perform basic support tasks for users within their own departments.

### Granular Attribute Delegation
Move from broad property permissions to more targeted controls (e.g., telephone number, office location, or other specific attributes).

### Expanded Object Coverage
Extend delegation to computer objects (including domain join rights) and security group management.

### Auditing and Oversight
Enable directory auditing on delegated OUs and establish periodic permission reviews to maintain long-term security hygiene.

## Strategic Direction
These planned expansions represent a natural progression from the current two-tier implementation. They will allow the lab to more closely simulate mature enterprise delegation practices while continuing to emphasize least-privilege principles and practical learning outcomes.

The existing infrastructure positions these enhancements for straightforward implementation.
